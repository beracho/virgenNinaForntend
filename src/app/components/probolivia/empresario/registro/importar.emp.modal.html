Importar archivo CSV
Pasos para importar el archivo CSV
  1. Descargue el archivo base haciendo click aquí.
  2. Proceda a llenar datos en el archivo proporcionado.
    Instrucciones de llenado
    1. Utilice siempre el archivo base proporcionado.
    2. No modifique la primera fila (cabecera).
    3. No aumente columnas solamente edite las casillas respectivas.
    4. La columna cantidad siempre debe ser un número entero sin decimales.
    5. La columna descripción y marca permiten valores alfanuméricos que no tengan un largo mayor a 100 caracteres.
    6. La columna precio unitario y total permiten ingresar valores numéricos con hasta dos decimales, siendo el punto (.) el caracter de separador de decimales.
  3. Guarde el archivo en formato “.csv” empleando como separador el símbolo tabulador.
  4. Adjunte el “archivo csv” en el área de abajo
Importante.- Compruebe que el archivo que desea importar NO se encuentre abierto.
